# CHaserClient
## このプログラムについて
このプログラムは、[CHaser](http://www.procon-asahikawa.org/contest.html)クライアントのC#用のライブラリです。<br />
このプログラムは**非公式**です。
## 公式ライブラリとの共通点・相違点
### 共通点
- サーバー側との通信内容。

### 相違点
- コンストラクタ内でキー入力を行わないようにした。
- 行動を指定する関数をGetReadyとそれ以外の2つにした。
- 行動の種類と方向の列挙型を作成した。
- サーバーとの通信のコードを簡単にした。
- ドキュメントコメントを追加した。
